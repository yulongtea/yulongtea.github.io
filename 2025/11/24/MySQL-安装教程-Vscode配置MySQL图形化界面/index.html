MySQL 安装教程+Vscode配置MySQL图形化界面

第一步：打开官网下载对应版本安装包

这只是一个Windows版本的安装器, 安装包里有64位的MySQL Server
https://dev.mysql.com/downloads/installer/

第二步：准备开始安装

  • 双击安装包开始安装
  • 然后选择自定义安装，因为有些应用我们可能用不到
  • 选择要安装的服务, 并点击箭头 将其添加到右边
  • 选中右边的产品，点击下方的Advanced Options更改文件夹
  • 可以根据自己的喜好选择安装目录
  • 接下来的配置默认即可。一直点击Next或者Execute直到输入密码
  • 输入你想要设置的密码即可，然后点击Next
  • 点击Next
  • 选择第三项，然后点击Next
  • 点击Execute
  • 之后点击Next和Finish退出即可

第三步：配置环境变量

  • 找到你安装在Program Files文件夹下的bin文件
    比如D:\applications\MySQL\Program Files\MySQL Server 8.0\bin
  • 打开系统属性，点击右下角环境变量
  • 点击系统变量的Path
  • 选择新建添加刚才的目录位置
  • 然后点击确定返回

第四步：检验是否安装成功

  • 打开cmd，输入以下内容

  • 打开Vscode，点击左侧插件栏位，搜索MYSQL并安装第二个MySQL和MySQL Syntax
